# proyecto_ajedrez
Proyecto para crear una página que de ajedrez.

Base de datos relacional (Postgresql):

- Usuario1, usuario2, con un booleano en código se crea una partida y solo al encontrar dos jugadores buscando se decide random si blancas y negras y se le atribuye un color, tmb con guardar un atributo como último tablero con su última jugada en la tabla de partidas, se puede guardar una partida si alguien quiere revisar una partida y volver para atrás se puede simplemente ir jugada a jugada de manera inversa para volver atrás.
